Release Notes Canigó 3.2

Darrera actualització: 20-12-2018

Release Notes

Canigó 3.2.7 (05/12/2018)

  • Autenticació amb certificat al servei de seguretat per GICAR
  • Gestió d’adjunts en servei de correu electrònic
  • Millores en parametrització connector d’Antivirus
  • Suport a tipus de dades JSON a PostgreSQL i MySQL
  • Revisió condicionals @ConditionalOnExpression al mòdul web.rs i security
  • Revisió nom dels fitxers que carrega els contextes de Spring als tests de canigo.security

Canigó 3.2.6 (28/09/2018)

  • Suport a base de dades MongoDB
  • Suport a base de dades PostgreSQL

Canigó 3.2.5 (18/06/2018)

  • Actualitzar Spring i altres llibreries
  • Reduïr el nombre de Threads que es crea a canigo.operation.instrumentation
  • El módul LOPD no interceptava GET/SET d’entitats
  • Configurar el filtre de QueryDSL per a que tracti els números com a Long
  • Suport a SAML2 per autenticació amb GICAR

Canigó 3.2.4 (05/04/2018)

  • No incloure per defecte els endpoints infoProperties i infoModules
  • Incorporar Bean mailSender a Canigo support mailing configurable
  • Ampliar l’endpoint de canigo.operation.instrumentation

Canigó 3.2.3 (20/12/2017)

  • Desenvolupament de jars amb test unitaris per als mòduls Canigó

Canigó 3.2.2 (29/09/2017)

  • Augmentar el logs existents en els mòduls de Canigó
  • Actualització Llibreries
  • Millores per donar suport a les millors pràctiques d’APIs REST

Canigó 3.2.1 (26/06/2017)

  • Actualització llibreries del mòdul SFTP
  • Fer configurable l’scan del Controller AuthController
  • Fer configurable el path d’on es carreguen les propietats al Servei de configuració

Canigó 3.2.0.1 (02/06/2017)

  • Actualització llibreries del mòdul SFTP

Canigó 3.2.0 (31/03/2017)

  • Actualització plantilla Canigó i plugin Eclipse
  • Suport REST i deprecar JSF
  • Incorporació noves llibreries
